CLI client#

mmkv-cli是mmkv提供给用户与mmkv-server基于命令行进行交互的CLI程序。

Command-line edit#

由于本CLI想要实现自动补全，历史记录，命令提示，清屏这些命令行编辑需求，因此依赖于linenoise。

Shell-command#

该CLI可以执行shell命令，只需要输入行以!开头，后面的输入内容会被视为shell命令及参数传递给/bin/shell等shell执行。
